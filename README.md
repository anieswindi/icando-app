# Project Title

ICANDO-App (Create React App)

### Installation

1. Clone Repo use HTTPS

```bash
git clone -b master https://github.com/anieswindi/icando-app
```

2. Use the node package manager to install this.

```
npm install
```

3. React start scripts

```
npm start
```
